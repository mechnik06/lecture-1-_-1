# Condition?
### Condition xdsh chiyay Condition xdsh ysloviya.
### Mo da JavaScript se namud Condition dorem.
### 1.if/else if/else.
```
if(2 == 2){
    console.log(true)
}
else if(2 == 3){
    console.log(false);
}
else{
    console.log("Not found");
}
```
### 2.Ternary operator.
```
  let num= 2 == 2 ? true : 3 == 2 ? false : "Not found";
```
### 3.Switch.
```
let dayOfWeek = new Date().getDay();
let message;
switch (dayOfWeek) {
    case 0:
        message = "Sunday";
        break;
    case 1:
        message = "Monday";
        break;
    case 2:
        message = "Tuesday";
        break;
    case 3:
        message = "Wednesday";
        break;
    case 4:
        message = "Thursday";
        break;
    case 5:
        message = "Friday";
        break;
    case 6:
        message = "Saturday";
        break;
    default:
        message = "Not found day";
}

console.log(message); 
```
